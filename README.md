# samsungRestore
Odin for macOS, free and downloadable
# Basics and how to install
It is odin for Mac, entirely in Xcode/Swift
Made using IOKit.usb, a Device searcher tool
Just click that .app (or .zip) file and extract it!
Insert bootloader, filesystem, and csc/cp
Very easy, uploaded to GitHub
# How to use
First get a tar
And download our file
And open our app and insert the tar in BL, AP, CSC, or CP
Click 'Flash OS' to flash
If you see any flaws fork it!
Wait while the device installs the firmware
And have fun!
We made it using vendor ID '0x04E8'
0x04E8 is the samsung vendor ID
# What does extracting mean?
Extracting means unzipping a zip or tar
Very easy to spell, letters: 'E-X-T-R-A-C-T-I-N-G'
